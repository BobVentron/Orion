Les forces et les limites d'Orion

✓
Ce qu'Orion fait mieux
  • Installation ultra-simple — 3 commandes, 5 minutes, zero configuration manuelle
  • Stack Docker clé en main — aucune dépendance à gérer sur le système hôte
  • Sécurité by design dès la v0.1 — non-root, secrets générés, isolation réseau
  • Schéma de données exhaustif conçu pour la roadmap complète — pas de refonte à prévoir
  • Export SVG natif de la cartographie pour les rapports d'audit
  • Fonctionne en air-gap total sans aucune dépendance cloud
  • CLI orion-control unifié — pas besoin de connaître Docker pour gérer l'instance
✗
Nos limites actuelles (v0.2-9)
  • Interface web encore en développement — les fonctionnalités UI arrivent en v0.3
  • Pas encore d'authentification utilisateur — JWT prévu en v0.3
  • LDAP / Active Directory planifié pour v0.4 seulement
  • Pas d'alertes email/webhook dans la version actuelle
  • Communauté et écosystème plus petits que LibreNMS ou Zabbix
  • Pas encore de signatures GPG sur le dépôt APT

Pas convaincu ? Testez vous-même.

Orion Community est gratuit, open-source et s'installe en 5 minutes. Le meilleur comparatif, c'est celui que vous faites vous-même sur votre propre infrastructure.
