4Protection des données personnelles

Données collectées par le logiciel Orion

Orion est un logiciel 100% on-premise. Toutes les données collectées par le logiciel (topologie réseau, inventaire des équipements, métriques de performance, adresses IP, adresses MAC) sont stockées exclusivement sur le serveur du client. Aucune donnée ne transite vers les serveurs de DuckInc.

Données collectées par ce site vitrine

Le site de présentation orion.duckinc.soleilhac.me ne collecte aucune donnée personnelle. Il n'y a :

  • Aucun cookie de tracking ou publicitaire
  • Aucun outil d'analytics tiers (Google Analytics, etc.)
  • Aucun formulaire de collecte de données personnelles
  • Aucun pixel de tracking ou beacon

Les logs serveur Nginx standards (adresse IP, user-agent, page visitée, horodatage) sont conservés 30 jours sur le serveur Web-Tabit pour des raisons de sécurité opérationnelle, puis supprimés automatiquement.

Responsable de traitement

Dans le contexte du logiciel Orion, le responsable de traitement des données réseau est l'organisation cliente qui installe et exploite le logiciel sur son infrastructure. DuckInc n'a accès à aucune donnée collectée par une instance Orion installée chez un client.

5Politique de confidentialité

Droit d'accès, de rectification et de suppression

Conformément au Règlement Général sur la Protection des Données (RGPD — Règlement UE 2016/679), vous disposez d'un droit d'accès, de rectification et de suppression des données vous concernant. Pour exercer ces droits, contactez l'équipe DuckInc à l'adresse indiquée en section 7.

Données des utilisateurs du logiciel

Les comptes utilisateurs créés dans une instance Orion (table app_users) sont gérés par l'administrateur de cette instance. Les mots de passe sont stockés sous forme hachée. DuckInc n'a aucun accès à ces données.

Transferts de données

Aucun transfert de données personnelles n'est effectué vers des pays tiers dans le cadre du fonctionnement d'Orion. Le dépôt APT hébergé sur OVH (France) ne contient que des fichiers binaires du logiciel — aucune donnée personnelle.
